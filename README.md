# Microservice Évènements

## Description
Ce microservice gère l’ensemble des fonctionnalités liées aux évènements.

Projet réalisé dans le cadre du cours **Applications Web Distribuées**.

## Responsabilités
- Création, modification et suppression d’évènements
- Gestion des catégories, capacités, tarifs, dates et lieux
- Publication et annulation d’évènements
- Recherche avancée multi-critères
- Recommandation d’évènements
- Analyse de popularité par catégorie

## Technologies
- Spring Boot
- Java 17
- MySQL

## Base de Données
- Base relationnelle assurant l’intégrité des données
