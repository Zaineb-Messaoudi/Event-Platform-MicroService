# Microservice Logistique

## Description
Ce microservice assure la gestion et l’optimisation des ressources logistiques des évènements.

Développé dans le cadre du cours **Applications Web Distribuées**.

## Responsabilités
- Gestion des équipements
- Gestion des salles et lieux
- Gestion du personnel
- Affectation des ressources aux évènements
- Vérification de disponibilité

## Technologies
- Spring Boot
- Java 17
- MySQL
