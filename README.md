# Microservice Blog

## Description
Ce microservice permet la gestion de contenu et l’interaction communautaire autour des évènements.

Développé dans le cadre du cours **Applications Web Distribuées**.

## Responsabilités
- Gestion des articles et publications
- Posts par utilisateurs et organisateurs
- Likes et commentaires
- Modération des contenus
- Détection de spam

## Technologies
- Spring Boot
- Java 17
- H2 (base embarquée)
