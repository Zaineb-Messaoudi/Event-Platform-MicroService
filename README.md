# Microservice Feedbacks & Réclamations

## Description
Ce microservice permet de collecter les avis des participants et de gérer les réclamations.

Projet réalisé dans le cadre du cours **Applications Web Distribuées**.

## Responsabilités
- Soumission et modification des feedbacks
- Système de notation (1 à 5 étoiles)
- Calcul de moyennes et statistiques en temps réel
- Gestion des réclamations et réponses des organisateurs

## Technologies
- Spring Boot
- Java 17
- H2 (base embarquée)
