# Microservice Abonnements

## Description
Ce microservice gère les abonnements et les stratégies de fidélisation.

Projet réalisé dans le cadre du cours **Applications Web Distribuées**.

## Responsabilités
- Abonnements utilisateurs (réductions, accès anticipé)
- Abonnements organisateurs
- Gestion des coupons et promotions
- Fidélisation intelligente

## Technologies
- Spring Boot
- Java 17
- MySQL
