# Microservice Réservations

## Description
Ce microservice gère le processus de réservation en garantissant la disponibilité des places en temps réel.

Développé dans le cadre du cours **Applications Web Distribuées**.

## Responsabilités
- Réservation sécurisée des tickets
- Vérification en temps réel des disponibilités
- Historique complet des réservations
- Annulation et remboursement automatisés
- Gestion de paiements sécurisé

## Technologies
- Spring Boot
- Java 17
- MySQL

## Points Clés
- Cohérence forte des données
- Gestion des pics de charge
- Traçabilité complète
